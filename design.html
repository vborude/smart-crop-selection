⚙️ System Design

User Input

The system collects user inputs such as soil type, temperature, water availability, irrigation method, and fertilizer type.

Processing Logic

A rule-based scoring system evaluates each crop based on matching conditions and assigns scores accordingly.

Output Result

The crop with the highest score is selected as the best crop, along with other suitable alternatives.

External Integration

The system connects users to government platforms for weather updates, market prices, and agricultural advisory.
